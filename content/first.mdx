今まで自分のポートフォリオになるページは GitHub Pages を使ってきた。
今の会社に入ってから内容の刷新をしておらず、jekyll をメンテする気持ちも起きなかったので、
勉強も兼ねて流行りの構成を組み立ててみた。

https://github.com/s4ichi/s4ichi.com

Next.js の 13 系を使って App Directory を試してみたり、Vercel にデプロイするフローを組んで触ってみたりと、
慣れないことばかりだったけど先駆者が大量に居たおかげである程度形にすることができた。

デザインやライブラリの選定などは基本的に Lee Robinson のサイトを参考にしていて、
構成や Next.js のプラクティスを学ばせてもらった。おかげで今はほぼ直感的に扱えるようになったので現状から色々変えていきたい。

https://github.com/leerob/leerob.io

Web サイトとしての完成度はまだまだ低くて、改善の余地が大量に残されているので片手間で改善しつつ、
こっそりモノを書く場が完成したので、技術に限らず色々書こうと思う。
